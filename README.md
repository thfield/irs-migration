## to get all the data
1. run `npm install`
1. run `./download.sh` (just county data)
  - or run `./download.sh all_data.txt` (for all state/county data)
1. (optional) run `./download.sh data_documentation.txt`
1. run `./geo.sh`
1. run `./parse-county.sh`
